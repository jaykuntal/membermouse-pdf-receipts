This plugin works in conjunction with MemberMouse. It sends an email receipt with PDF attachment on initial and rebill payments.

== Description ==

This plugin allows you to configure an email and PDF template that contains details relating to a purchases. The email will be sent to customers with a PDF attachment based on these template when an initial or rebill purchase happens. You can also resend past receipts based on an order number. 

Here's a link to the full [support documentation](https://support.membermouse.com/support/solutions/articles/9000197357-send-pdf-receipts-to-members "MemberMouse PDF Receipts documentation") which provides in-depth instructions for how to configure and use this plugin.

== Frequently Asked Questions ==

= Do I need MemberMouse to use this plugin? =

Yes. You need to have MemberMouse 2.4.0 or above installed and activated to use this plugin.
